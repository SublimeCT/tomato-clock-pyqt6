🍅 番茄专注
把注意力留在任务上

番茄专注（TomatoClock）是一款跨平台桌面番茄钟，但它更在意“过程”而不是“计时”。从开始到结束，你只需要做更少的选择：启动即进入专注，进度常驻状态栏/托盘，统计用来复盘节奏与规律。支持 Windows / MacOS / Linux

PyQt6 · QWidget Windows · 桌面 macOS · 桌面 Linux · 桌面
启动更干净一键进入

把关键动作放到最短路径：打开就能开始，不让设置与选择偷走第一分钟。

过程更克制少打断

用状态栏/托盘承载进度，让你不必“回去看一眼”，从而减少上下文切换。

反馈更可信可复盘

用日历与年度视角记录投入，帮助你找到高效时间段与真实节奏，而不是凭感觉猜。

从计时工具到专注系统

真正的损耗往往来自三件事：开始前犹豫、进行中切换、结束后没有反馈。番茄专注把它们做成一条更顺的路径，让专注更像习惯而不是任务。

启动：一键进入低摩擦

把关键动作放在最短路径：不让“先设置一下”消耗掉开始专注的动量。

进行：进度外置少切换

状态栏/托盘承载倒计时：你不需要回到应用里确认剩余时间，注意力持续停留在当前任务。

复盘：节奏可见有依据

用日历/年度视角看见趋势：什么时候高效、什么时候需要休息，让策略有据可依。

关键界面（用于理解，不占主角）

每张截图只限制宽度、保持原始宽高比自适应高度；不提供点击放大；统一结构：左图右文，保持样式一致。

Focus

专注

把“下一步做什么”放在最醒目处：计时、状态、切换动作形成一条线，不让你在界面里找路径。

开始动作更明确，减少犹豫。状态信息集中呈现，减少扫视成本。
Stats

统计

用一眼能读懂的结构呈现趋势：你不是在“看图”，而是在确认自己是否在正确的节奏里。

重点优先：先趋势，后细节。信息分组清晰，减少视觉噪音。
Year

年度

把“我到底有没有坚持”变成可回答的问题：年度视角更适合看阶段性成果与季节性波动。

更长周期更接近真实节奏。让复盘从“感觉”变成“依据”。
Settings

设置

设置不是“配置中心”，而是为了让你更少维护：把会影响体验的选项放在合理的位置，避免打断主流程。

重要项优先：少而清晰。一致的控件节奏与反馈。
Tray

状态栏 / 托盘

把进度放在“视线边缘”：你专注在工作时，不需要回到应用里确认倒计时。

更少切换，更少中断。进度可见但不侵占注意力。
